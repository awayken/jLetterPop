jLetterPop Demo

Welcome to the jLetterPop demo page! You're probably here to see jLetterPop in action, so let's get started.

Pop These Letters

See the project on Github, or download a zip file of the project.

The script is pretty simple at this point. Just call the plugin on whatever element you want.

<script src="jquery.letterpop.min.js"></script> <script> $('.letterpop').letterpop(); </script>

The plugin uses the following CSS classes:

  • .letterpop - added to the element you call the plugin on
  • .letterpop_letter - each letter gets a span with this class
  • .letterpop_popper - the animating letter is a span with this class
  • .letterpop_hover - class gets add to each letter when you hover
